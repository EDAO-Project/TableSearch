The task is to find tables that are semantically relevant to a table query.

You will be presented with a tabular query containing DBpedia entities and a set of tables from Wikipedia.
The task is to annotate these tables according to their semantic relevance to the table query.

As an example, a table query containing american presidents and the state they are born in is relevant to tables describing presidents, prime ministers, politicians, celebreties, etc., as well as tables describing cities, states, countries, geographical localtions, etc.
Therefore, a table does not need to contain keyword matches with the query in order to be semantically relevant.
