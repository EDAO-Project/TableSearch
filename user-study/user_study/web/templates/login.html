Instructions

You will be presented with a tabular query and a set of tables from Wikipedia.
The task is to annotate the tables with a relevance score between 1 and 4 given a table query based on the tables' semantic relevance to the query.
For example, a table query containing american presidents and the state they are born in is relevant to tables describing presidents, prime ministers, politicians, celebreties, etc., as well as tables describing cities, states, countries, geographical localtions, etc.
Therefore, a table does not need to contain keyword matches with the query in order to be semantically relevant.
Some examples of relevant and irrelevant tables are shown in the bottom of this page.

To login, use a name you like and remember it.
If you choose to stop annotating tables and return later, simply use the same username to login again.
